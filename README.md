# compare_activity
A page to compare running activities.


GPX Activity Compare

A single-page, fully client-side tool for comparing two or more Garmin GPX
activities side by side: heart rate, cadence, pace, elevation, and route map,
with a linked X-axis and a draggable marker that stays in sync across charts.

Nothing is uploaded anywhere — GPX files are read directly in your browser
and never leave your device. The whole app is one HTML file with no external
dependencies (no CDN, no fonts, no API calls), so it also works if you just
double-click index.html locally, with no server or internet connection.
